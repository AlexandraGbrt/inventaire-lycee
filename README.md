# 🏫 Gestionnaire d'Inventaire Lycée

Ce projet permet de gérer le parc informatique (PC Portables et Imprimantes).

## 🏗️ Architecture
- **parc-api** : Serveur TypeScript (Back-end) gérant les données.
- **parc-client** : Interface Web (Front-end) pour le gestionnaire.

## 🚀 Installation
1. Aller dans chaque dossier (`parc-api` et `parc-client`).
2. Lancer `npm install` pour installer les outils.

npx ts-node src/index.ts
http://localhost:3000/api-docs
